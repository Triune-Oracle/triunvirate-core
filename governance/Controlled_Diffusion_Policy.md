# Controlled Diffusion Standard (CDS)

## Core Rule
Information may only propagate with guarantees it can support.

## Diffusion Layers

### Agent Layer
- **Allowed**: DSP structure, role limits
- **Disallowed**: Outcome guarantees, financial claims

### Code Layer
- **Allowed**: Prompts, metrics, logs
- **Disallowed**: Symbolic or narrative language

### Documentation Layer
- **Allowed**: Metaphors (clearly labeled)
- **Disallowed**: Unverified assertions

### External / Marketing Layer
- **Allowed**: Framing principles
- **Disallowed**: Inevitability or promises

### Financial Layer
- **Allowed**: Contract logic only
- **Disallowed**: Narrative framing

## Enforcement
If a statement cannot be audited, it does not propagate.
